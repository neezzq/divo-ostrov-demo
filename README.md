# Диво Остров — сайт парка аттракционов

Многостраничный сайт парка аттракционов «Диво Остров» (Крестовский остров,
Санкт-Петербург). Премиально-минималистичный дизайн, «дорогие» анимации,
колесо обозрения с каталогом аттракционов. Без сборки — чистый HTML/CSS/JS.

## Страницы

| Файл | Назначение |
|------|------------|
| `index.html` | Главная: колесо аттракционов (кабинки = аттракционы) |
| `attractions.html` | Каталог всех аттракционов с фильтром |
| `attraction.html?a=slug` | Страница аттракциона (данные по slug) |
| `visit.html` | Как добраться, режим работы, контакты |

## Архитектура данных («бэкенд»-слой)

Все аттракционы описаны в одном источнике — `assets/js/data.js` (`window.DIVO`).
Каталог и страницы аттракционов рендерятся из него на лету:

```
assets/js/
  data.js     — данные аттракционов + API (bySlug, byCategory…)
  site.js     — общий каркас: шапка, подвал, меню, переходы, анимации
  home.js     — колесо обозрения на главной
  catalog.js  — сетка каталога + фильтр
  detail.js   — страница аттракциона по ?a=slug
```

Чтобы добавить аттракцион — допишите объект в `data.js`, страница появится сама.

## Дизайн и анимации

- Шрифты Cormorant (дисплейный сериф) + Manrope
- Палитра: тёплая бумага · графит · красный акцент
- Колесо обозрения с вертикально стабилизированными кабинками
- Плавные переходы между страницами, scroll-reveal, кастомный курсор
- Адаптив, поддержка `prefers-reduced-motion`

## Запуск

```bash
python3 -m http.server 8000   # http://localhost:8000
```
