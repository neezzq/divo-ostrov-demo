# Диво Остров — сайт парка аттракционов

Современный демонстрационный сайт для крупнейшего парка аттракционов
Санкт-Петербурга «Диво Остров» (Крестовский остров). Информация о парке
взята с официального сайта [divo-ostrov.ru](https://divo-ostrov.ru) и открытых
источников, дизайн полностью переработан.

## Возможности

- Анимированный hero с градиентным фоном и счётчиками
- Каталог аттракционов с фильтром по категориям (экстрим / семейные / детские)
- Блоки цен и безлимитного браслета
- Раздел «Как добраться» с картой и режимом работы
- Адаптивная вёрстка, бургер-меню, scroll-reveal анимации
- Чистый стек: HTML + CSS + ванильный JavaScript, без сборки

## Запуск

Откройте `index.html` в браузере или поднимите статический сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Структура

```
index.html
assets/
  css/styles.css
  js/main.js
```

## Информация о парке

- **Адрес:** Санкт-Петербург, Кемская ул., 1, Крестовский остров
- **Режим работы:** будни 11:00–21:00, выходные 11:00–22:00 (зимой закрыт)
- **Вход:** свободный, оплачиваются только аттракционы

> Демо-проект. Цены и часы работы ориентировочные — уточняйте на официальном сайте.
